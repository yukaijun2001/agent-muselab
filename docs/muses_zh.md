# 九位缪斯

> [English](muses.md)

muselab 命名源自希腊神话中的**九位缪斯**——艺术与学问的守护女神。**Muse** 是运行于其中的 AI 人格；**muselab** 是承载她的工作坊。

首次访问时随机分到一位缪斯，之后保持不变，直到你点击顶部的吉祥物图标手动切换。网站图标和浏览器标签页随之同步。

| 缪斯 | 领域 | 几何形 |
|---|---|---|
| Calliope（卡利俄佩） | 史诗 | 六边形 |
| Clio（克利俄） | 历史 | 层叠横线（史册堆叠）|
| Erato（厄拉托） | 情诗 | Vesica piscis（双圆相交）|
| Euterpe（欧忒耳佩） | 音乐 | 声波 |
| Melpomene（墨尔波墨涅） | 悲剧 | 残月 |
| Polyhymnia（波吕许谟尼亚） | 圣诗 | 圣光环 |
| Terpsichore（忒耳普西科瑞） | 舞蹈 | 三点呼应（三角形）|
| Thalia（塔利亚） | 喜剧 | 火花 |
| Urania（乌拉尼亚） | 天文 | 行星轨道 |
