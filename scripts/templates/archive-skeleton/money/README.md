# money/

跟钱有关的一切 —— 预算、收支、持仓、学贷、保险、退休规划。

## 放什么

- 月度预算 / 收支记录
- 持仓表（账户、品种、比例）
- 贷款情况（学贷 / 房贷 / 信用卡）
- 保单（重疾 / 医疗 / 寿险 / 财产）
- 退休账户 / FIRE 计算
- 报税材料（自由职业 / 自雇）

## Muse 在这里的约束

- 引用学术与一手数据（Fama-French / Bogle / Bloomberg / Morningstar 等），不引情绪化自媒体
- 调研标的时给出：跟踪指数、费率、规模、流动性、税收结构、历史回撤、与现有持仓相关性
- 不预测「会涨 / 会跌」
- 不推荐期权、杠杆 ETF、meme 股、借钱加杠杆投资
- 学生 / 低收入阶段：**省钱**优先于**理财收益**
- 大额决策（买房 / 留学 / 换城市）列出前置假设及在什么条件下应重新评估，不替你拍板

## 注意

- 数字可以脱敏（写「约 X 万」），Muse 用大致范围即可
- 不要把账户密码 / API key / 银行卡号写在档案里
- 历史决策的「为啥这么做」比「做了啥」更重要
