# work/

工作与产出的记录 —— 学业、工作、自由职业、创业、照护、退休活动。

## 放什么

- 简历 / 作品集 / 学业材料 / 论文 / 申研材料
- 项目复盘、绩效记录、面试笔记、目标公司清单
- 客户档案、合同、项目交付物（自由职业）
- 商业计划、产品文档、招聘 / 融资材料（创业）
- 育儿 / 长辈照护日志（全职照护）
- 当前主要活动的记录（退休 / Gap）

## Muse 在这里的约束

- 简历 / 文书：STAR + 量化（结果带数字），动词开头，禁用「负责」
- 不虚构经历、不夸大影响，级别描述跟实际匹配
- 谈判 / 申请只推荐能写进合同 / 录取条件的硬约束（Sign-on、奖学金金额、入学条件），不依赖 HR / 招生口头承诺
- 涉及他人的内容（同事 / 客户 / 教授）脱敏处理

## 注意

- 文档命名包含日期 `YYYY-MM-xxx.md/pdf`
- 学习 / 工作笔记记录「为什么这么做」的判断依据，比「做了什么」更有价值
