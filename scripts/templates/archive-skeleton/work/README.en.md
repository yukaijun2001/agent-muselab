# work/

Active work artifacts — coursework, employment, freelance, founding,
caregiving, retirement activities.

## What goes here

- Resume / portfolio / study materials / papers / grad-school applications
- Project retrospectives, performance records, interview notes, target-company list
- Client files, contracts, deliverables (freelance)
- Business plan, product docs, hiring / fundraising materials (founding)
- Childcare / eldercare logs (full-time caregiving)
- Current activity notes (retirement / gap)

## Muse's constraints here

- Resume / docs: STAR + quantification (results with numbers); verbs first;
  never "responsible for"
- No fabricated experience; no exaggerated impact; level descriptions match
  actual scope
- Negotiation / application: only pushes for things that can be written into
  the contract / admission letter (sign-on, scholarship amount, start
  conditions); doesn't gamble on HR / admissions verbal promises
- Redact references to other people (colleagues / clients / professors)

## Notes

- Date-prefix filenames: `YYYY-MM-xxx.md/pdf`
- Notes that capture "why this decision" outlast notes that capture "what was done"
