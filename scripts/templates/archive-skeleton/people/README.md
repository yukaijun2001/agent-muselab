# people/

关于身边重要的人的笔记 —— 伴侣、父母、子女、室友、长期朋友、导师、宠物等。

## 放什么

- 每个重要的人一份简短笔记（年龄 / 健康 / 阶段 / 关系动态）
- 家庭决策（照护安排 / 教育方案 / 家务分工）
- 伴侣 / 婚姻笔记（共同目标 / 近期重要事项）
- 父母 / 长辈健康清单、保单、重要日子
- 子女成长记录（如有）
- 重要日期（生日 / 纪念日 / 手术日 / 移居日）

## Muse 在这里的约束

- 跨领域决策时把多个维度信息**串起来**（如「父母手术 + 你今年现金流 + 你的休假天数」一起评估）
- 重大决策列正反、列前置假设、列在什么条件下应重新评估
- 不替你拍板，列出选项 + 各自代价

## 注意 — 这是最敏感的目录

- 强烈建议给整个 muselab archive 启用文件系统加密（macOS FileVault / Linux LUKS）
- 不要 sync 到公共云（OneDrive / Google Drive / Dropbox）
- 涉及他人的信息可用「M」「父亲」「室友 A」代替真名
- 不要写社保号 / 身份证号 / 银行账号 — 这些应存放在密码管理器中
- 离异 / 法律纠纷材料：考虑单独加密
