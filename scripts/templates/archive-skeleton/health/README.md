# health/

身体相关：体检报告、药、补剂、训练记录、长期健康问题。

## 放什么

- 体检报告 PDF（命名 `YYYY-MM-体检-机构.pdf`）
- 当前在用的药 / 补剂（剂量、品牌、买入时间、服用原因）
- 运动 / 训练记录
- 睡眠 / 心率 / 步数导出
- 就医记录、过敏、慢性病
- 家人的健康（父母 / 子女 / 伴侣）按子目录分人放

## Muse 在这里的约束

- 不下诊断、不开处方；涉及重要决定时提醒就医
- 引用主流指南（AHA / ADA / USPSTF / 国内一级学会等），不引自媒体
- 给数字时带单位、剂量、时间窗、副作用、禁忌
- 不推 MLM 保健品，不夸大补剂效果
- 数字原样保留 — Muse 会引用具体值
- 写清单位（mmol/L vs mg/dL、ng/mL vs nmol/L）

## 注意

- 病案号 / 身份证号脱敏后再放
- 心理健康 / 性健康 / 精神类用药：考虑单独加密文件
