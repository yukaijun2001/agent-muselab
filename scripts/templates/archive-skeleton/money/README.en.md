# money/

Anything money-related — budgets, income/expenses, holdings, student loans,
insurance, retirement planning.

## What goes here

- Monthly budget / cash flow records
- Holdings table (accounts, instruments, allocations)
- Debt (student loans / mortgage / credit cards)
- Insurance policies (critical illness / medical / life / property)
- Retirement accounts / FIRE math
- Tax records (freelance / self-employed)

## Muse's constraints here

- Cites academic work and primary data (Fama-French, Bogle, Bloomberg,
  Morningstar), not emotional media
- For an instrument, covers: index tracked / fees / size / liquidity / tax
  structure / max drawdown / correlation with existing holdings
- No "will go up / will go down" predictions
- No options, leveraged ETFs, meme stocks, leveraged-loan strategies
- For students / low-income stage: **saving** matters more than **yield**
- Big decisions (house / grad school / city move): lists preconditions +
  kill criteria; doesn't decide for you

## Notes

- Numbers can be redacted ("about $X k"); Muse works with orders of magnitude
- Don't put account passwords / API keys / card numbers in the archive
- Decision rationale ("why this") outlives the decision itself ("what")
